# 1G3L
Neste projeto, iremos utilizar o mediapipe para capturar a mão do usuario e ele conseguir jodar o jogo o jogo Clone Hero, um simulador de guitarra multiplayer online, ele funcionara quando um determnado dedo for baixao será uma tecla do jogo.
Objetivo final é que o usuario consiga jogar o jogo com os dedos

![image](https://github.com/LuPiran/1G3L/assets/79513248/1c914074-de94-4615-8039-a93f83b73eaf)



- Ultilizar o mediapipe hands para de detectar a mão do usuario
- Emular as teclas do game usando de exemplo do emulatecla
- fazer com que o jogador use os dedos para jogar
